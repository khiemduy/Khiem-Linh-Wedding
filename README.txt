Wedding template - how to use
----------------------------
Files:
  - index.html : main page
  - css/style.css : styles
  - js/script.js : basic interactions
  - data/config.js : edit this file to change names/date/place/album/music
  - assets/images/ : put your hero.jpg and album images here (replace placeholders)

Quick start:
  1) Open the folder in VS Code.
  2) Install Live Server extension and "Open with Live Server" on index.html.
  3) Edit data/config.js - change names, date (ISO format), place, heroImage, album array, music.
  4) Replace files in assets/images/ with your images.
  5) Commit & push to GitHub and enable GitHub Pages (branch main / root).
